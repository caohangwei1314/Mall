## 介绍
本项目是基于 SpringBoot + Dubbo 搭建而成的分布式秒杀系统，系统模块包括：用户模块、商品模块以及前端模块。项目结合 [Java秒杀系统方案优化 高性能高并发实战](https://coding.imooc.com/class/168.html) 以及开源项目 [zheng](https://github.com/shuzheng/zheng) 实现。
